# Week 1 Lab: Inspecting and Manipulating Digital Audio

## Learning goals

By the end of this lab, you should be able to:

- load WAV files in Python
- inspect sample rate, shape, and duration
- plot waveforms
- create lower-resolution versions of audio
- compare signals by listening and visualization
- describe what changes when sample rate or bit depth are reduced

## Files used in this lab

Recommended input files:

- `assets/audio/speech.wav`
- `assets/audio/music.wav`
- `assets/audio/sine_440hz.wav`

Write short answers to these questions:
